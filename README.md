## Wolf3d
* *A wolf3d like game, made with raycasting*

## demo
<img src="ressources/img/wolf3d_demo.gif" width="400"></img>

## Built with

C language</br>
SDL2 library</br>
Raycasting method</br>

## Installing

* Git clone http://github.com/sgalasso/wolf3d.git
* Compile it with `make`

## Usage

`./wolf3d path/[map]`

## How to create a valid map

* create map file
* *first line* [w] [h]
* builting:
* *0 : void*
* *1 : wall*
* *2 : player spawn*
* *separate with space*

## Map exemple

```
4 4
1 1 1 1
1 0 0 1
1 2 0 1
1 1 1 1
```

## How to use it

* show config `i`
* changing mode :
* *`1` normal*
* *`2` gaming*
* *`3` dev*
